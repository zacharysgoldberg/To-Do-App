# ToDos

## Description

To-do list application.

## Design

FastAPI backend, bootstrap frontend, PostgreSQL, SQLAlchemy, Alembic, Nginx, Docker, and Kubernetes manifests.

## Features

Reset password, ToDo reminder notifications, and JavaScript FullCalendar CDN.
